❓ ST APPS ヘルプ

🔐 ログインについて
2026-07 からすべてのアプリで Googleログイン が必要になりました（ふたりのGmailだけが許可されています）。 端末ごとに初回1回ログインすれば、以後は自動です。HUBでログインすると、各アプリの「あなたはどっち？」も自動でスキップされます。

💰 CASHFLOW 開く →

フリーランス収支・経費・税金管理。月次手取り・年間税額シミュレーション、売掛（入金待ち）管理。RINGIで承認された経費が自動で反映される。

🧾 SE.TSU 開く →

ふたりの家計簿。実支出の記帳・カテゴリ予算・共同財布の50/50割り勘精算。RINGI承認・WISHLIST購入分の取込インボックス、CSVインポート対応。

📝 RINGI 開く →

購入申請・決裁システム（家庭内稟議）。申請 → 相手が承認/否決/保留/条件付き承認。承認された経費はCASHFLOW・SE.TSUに連携。購入後レビュー機能つき。

🛍️ WISHLIST 開く →

欲しいものリスト。アイテムからワンタップでRINGI申請、承認されるとステータス自動更新。ふたりのアイテム比較画面あり。

🧺 STOCK NEW 開く →

買い物リスト＋日用品在庫。「切れそう」「買う」をふたりで共有し、買ったらワンタップでSE.TSUに記帳。定番品は在庫ローテーションで管理。

📅 DAYS 開く →

共有カレンダー。月/週表示・祝日・記念日カウントダウン・スタンプ・ToDo・テンプレート。CASHFLOWの入金/固定費/税期限、RINGI期限も自動表示。

⏱ Sche Queue 開く →

Googleカレンダーへの予定一括投入ツール（発車標風UI）。個人のGoogleアカウントで動作。

📊 Gantt 開く →

ガントチャート型タスク管理（Vercel）。Googleカレンダー連携。共同作業者向けの公開版は GANTT_PUBLIC。

🎯 GOALS 開く →

ふたりの目標・貯金ゴール管理。CASHFLOWの貯蓄残高と連携できる。

💝 Konomi 開く →

好み相性データベース（好き/まあ好き/苦手/NG/条件付き × 7カテゴリ）。早見表・ごはん提案機能つき。プレゼント・お店選びで外さないための辞書。

📒 Hobby Ledger 開く →

趣味の活動時間・支出の記録。ストリーク・月間グラフ・ふたり比較・年間サマリー。

🔔 通知のしくみ

  • HUBのベル: 各アプリからのアプリ内通知（30日で自動削除）
  • ntfyプッシュ: スマホの「ntfy」アプリで共有トピックを購読すると即時通知が届く（トピック名はRINGIの設定画面参照）
  • 週次サマリー: 毎週日曜9時に1週間のまとめが届く
